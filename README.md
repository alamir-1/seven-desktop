# SEVEN Desktop App

تطبيق سطح المكتب (Windows + Mac) لموقع SEVEN، مبني بـ Electron.

## الفكرة باختصار
التطبيق ده مجرد "نافذة" بتفتح موقعك (https://alamir-1.github.io/SEVEN-27-3/) بشكل تطبيق مستقل بأيقونة واسم خاص بيك، بدون شريط عنوان متصفح، ومع دعم تحميل الملفات (إكسل/PDF/صور) مباشرة لمجلد التنزيلات.

## خطوات الحصول على التطبيق الجاهز (.exe و .dmg)

### 1) ارفع المشروع على GitHub
- اعمل Repository جديد على GitHub (مثلاً اسمه `seven-desktop`)
- ارفع كل الملفات والمجلدات الموجودة هنا بداخله (بما فيها مجلد `.github` المخفي)

### 2) البناء هيبدأ تلقائياً
بمجرد ما ترفع الملفات، GitHub Actions هيشتغل لوحده تلقائياً (مفيش زرار تدوسه):
- روح تبويب **Actions** في صفحة الـ repo بتاعك
- هتلاقي عملية بناء شغالة اسمها "Build SEVEN Desktop App"
- استنى لحد ما تخلص (حوالي 5-10 دقايق، وبتبني نسخة ويندوز ونسخة ماك في نفس الوقت)

### 3) تحميل التطبيق الجاهز
- لما البناء يخلص (علامة صح خضرا ✅)، دوس على اسم عملية البناء
- انزل تحت لقسم **Artifacts**
- هتلاقي:
  - `SEVEN-windows-latest` → فكه هتلاقي جواه ملف `.exe` (مثبّت لويندوز)
  - `SEVEN-macos-latest` → فكه هتلاقي جواه ملف `.dmg` (مثبّت للماك)
- حمّل الملف المناسب لجهازك وثبّته زي أي برنامج عادي

## ملاحظات
- التطبيق بياخد آخر تحديث من موقعك تلقائياً في كل مرة تفتحه (مفيش كاش قديم)
- أي رابط خارجي (زي واتساب أو فيسبوك لو موجود) هيتفتح في متصفح الجهاز مش جوه التطبيق
- لو غيّرت رابط الموقع في المستقبل، هتحتاج تعدّل السطر `SITE_URL` في ملف `main.js`
